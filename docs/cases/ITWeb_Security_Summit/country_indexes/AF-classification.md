# Africa (AF) — Hybrid Cyber–Influence Incident Classification

This table classifies 13 incidents targeting African entities according to the cyber–influence taxonomy:
- **(1) Cyber-enabled influence** — cyber operations serving an influence goal (1a–1h)
- **(2) Influence-enabled cyber** — disinformation/deception enabling a cyber intrusion (2a–2h)
- **(3) Parallel cyber–influence** — simultaneous, independent cyber and influence operations by the same actor

Where an incident is predominantly a coordinated-inauthentic-behaviour (CIB) campaign built on cyber-enabled fake-account / fake-site infrastructure but does not map cleanly to a numbered sub-type, it is classified under **1c (broad reading: injecting propaganda via cyber-enabled fake assets and impersonation infrastructure)**, which best reflects the taxonomy's framing of "injecting propaganda or disinformation into websites".

## Continental observations

- **Dominant pattern:** cyber-enabled influence via CIB infrastructure (type 1c) — 11 of 13 incidents.
- **Dominant perpetrator profile:** Russian/Russian-linked actors and domestic political/PR networks, with cybercriminal extortion groups responsible for the two ransomware-driven cases.
- **Two clear double-extortion (2f) cases:** Kenya Airways and the South African industrial-sector wave — both leverage leak-site doxxing as the coercive lever.
- **Most-affected victim categories:** elections and voters, civil society, and (in the ransomware cases) private-sector / industrial organisations.
